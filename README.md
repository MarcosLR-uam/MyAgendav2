# MyAgendav2
“MyAgenda” es un sistema de escritorio que permite al usuario registrar, visualizar, editar y eliminar tareas personales. Cada tarea incluye información como título, descripción, fecha límite, estado, etc. La aplicación implementa Programación Orientada a Objetos, se conecta a una base de datos PostgreSQL.
